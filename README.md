# LocalTime
Инициализируй шесть статических переменных:

в timeMax запиши максимально возможное время;
в timeMin запиши минимально возможное время;
в time1 запиши время 14 часов ровно;
в time2 запиши время 7 часов 8 секунд;
в time3 запиши время 0 часов 45 минут 61 наносекунда;
в time4 запиши время 14 часов 45 секунд 1001 наносекунда.
В методе main() создай объект LocalTime, используя часы из time1 (time1.getHour()), минуты из time2, секунды из time3, наносекунды из time4. Выведи его в консоль.



Псевдокод:

```
ПРОГРАММА ТренировкаНаВремя

ИМПОРТИРОВАТЬ java.time.LocalTime

ОБЪЯВИТЬ:
    timeMax = LocalTime.MAX              // Максимальное время (23:59:59.999999999)
    timeMin = LocalTime.MIN              // Минимальное время (00:00:00.000000000)
    time1 = LocalTime.of(14, 0)          // 14:00:00.000000000
    time2 = LocalTime.of(7, 0, 8)        // 07:00:08.000000000
    time3 = LocalTime.of(0, 45, 0, 61)   // 00:45:00.000000061
    time4 = LocalTime.of(14, 0, 45, 1001)// 14:00:45.000001001

НАЧАТЬ
    // Создать новое время, комбинируя компоненты из разных переменных:
    // - Часы взять из time1 (14)
    // - Минуты взять из time2 (0)
    // - Секунды взять из time3 (0)
    // - Наносекунды взять из time4 (1001)
    
    новоеВремя = LocalTime.of(
        time1.получитьЧасы(),     // = 14
        time2.получитьМинуты(),   // = 0
        time3.получитьСекунды(),  // = 0
        time4.получитьНаносекунды() // = 1001
    )
    
    ВЫВЕСТИ новоеВремя  // Результат: 14:00:00.000001001
КОНЕЦ
```

Пояснение:

Программа создает новое время, комбинируя компоненты из четырех разных временных объектов:

· Часы: 14 (из time1)
· Минуты: 0 (из time2)
· Секунды: 0 (из time3)
· Наносекунды: 1001 (из time4)

Итоговое время: 14:00:00.000001001
